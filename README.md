# Init
SCP: Classified Site file for plugins on lua

Файл, который необходимо установить, чтобы плагины на луа, которые находятся здесь, работали. 
Также он добавляет комнату админов (если установлен bundle файл).

**Инструкция по установке:** файл `init.lua` поместить в директорию с обычными плагинами. Папки из архива `client.zip` извлечь в папку `client`. Файл из папки `server ([ОС сервера])` извлечь в папку `server`. Таким образом, должно получиться так:
```
Plugins
|-init.lua
|-client
| |-Android
| | |-adminroom
| |-Windows
| | |-adminroom
|-server
  |-adminroom
```

**Важное примечания:** название файла менять нельзя (конкретно этот файл должен быть назван *init.lua*). Также и со всеми плагинами на луа. Также рекомендуется установить все плагины (особенно на кастом. класс ПХ), чтобы избежать ошибок.
